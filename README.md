# GNY Blockchain

## 1 Install

Clone this repository:
```bash
git clone https://github.com/gnyio/gny-experiment
```

## 2 Install Dependencies

Execute:
```bash
npm install
```

## 3 Transpile Files with TypeScript

Execute:
```bash
npm run tsc
```

## 4 Start Blockchain

Change directory to the `dist` dir and start the Blockchain:
```

cd dist
node app
```
